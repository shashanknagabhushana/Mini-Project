# Resume Screening and Application Tracking System

## Project Overview
This project is a web-based Resume Screening and Application Tracking System designed to help recruiters manage candidate applications efficiently.

## Features
- Add Candidate Details
- Resume Skill Screening
- Match Percentage Calculation
- Application Status Tracking
- Dashboard Analytics
- Local Storage Support
- Clear All Data Option

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage

## How to Run
1. Download the project files.
2. Open index.html in a web browser.
3. Add candidate details.
4. Enter required skills.
5. Click "Screen Candidates" to evaluate applicants.
