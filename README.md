# Resume Screening & Application Tracking System

## Project Overview

The Resume Screening & Application Tracking System is a web-based recruitment management application developed using HTML, CSS, and JavaScript. The system helps recruiters manage candidate information, perform skill-based screening, track candidate status, generate recruitment insights, and download recruitment reports.

The project aims to simplify the recruitment process by automating candidate evaluation based on required skills and providing an organized platform for application tracking.

## Project Workflow

1. Recruiter enters candidate details.
2. Candidate information is stored in browser local storage.
3. Required skills are entered for screening.
4. The system compares candidate skills with required skills.
5. Match percentage is calculated automatically.
6. Candidates are classified as:
   - Applied
   - Shortlisted
   - Rejected
7. Dashboard analytics are updated.
8. Recruitment recommendations are generated.
9. Reports can be downloaded for further review.
